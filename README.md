# cafe-sample — HARUCHI 하루치 베이커리

이피웹(YIPI WEB) 포트폴리오용 ④번 샘플. 감성 카페/베이커리.

- **매트릭스 셀:** 단일앵커 · **메뉴-퍼스트** · **좌우 분할** · 스티키 앵커 내비 · 이벤트 팝업+인스타 피드+지도 · **레트로 플레이풀(딥그린+크림+머스타드)**
- **차별점:** 기존 3종(예약/포폴/서사)과 달리 "메뉴가 주인공". 첫 레트로/플레이풀 무드 + 첫 좌우 분할 레이아웃. 베이지 팔레트 탈출 계열(딥그린).
- **타입:** Fraunces(레트로 디스플레이 세리프) + Pretendard.
- **키커 변주:** 영문 지역명 대신 한글 키커("서울 망원 · 동네 빵집") 사용.
- **연락:** 네이버 예약(외부 링크) + 인스타 + 전화. 대관·단체는 별도 "문의하기" 폼(Formspree). 모바일 하단바 = 전화·네이버예약·인스타(PC CTA와 통일).

배포: Cloudflare Workers (정적 자산). 폼은 Formspree `YOUR_FORM_ID` 치환 필요.
